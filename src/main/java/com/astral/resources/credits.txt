
  Some features of Astral Invoice rely on external libraries here is a list of them:

  1. SQLite JDBC Crypt - https://github.com/Willena/sqlite-jdbc-crypt
  2. FlatLaf - https://www.formdev.com/flatlaf
  3. OpenPDF - https://github.com/LibrePDF/OpenPDF
  4. ICU4J (International Components for Unicode) - https://icu.unicode.org
